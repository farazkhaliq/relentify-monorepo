## What this does

Lets you attach files such as receipts, supplier invoices, and supporting documents to bills, expenses, and bank transactions. Attachments are stored alongside the record so you can find them during audits or VAT inspections without digging through emails.

## When to use it

Attach a file whenever you receive a receipt, supplier invoice, or any document that supports a transaction. This is especially useful for expense claims, where HMRC may require proof of purchase.

## Step by step

1. Open the bill, expense, or bank transaction you want to attach a file to
2. Scroll down to the **Attachments** section
3. Click **Upload** or drag and drop your file into the upload area
4. Select the file from your device — supported formats are PDF, JPEG, PNG, and WEBP
5. The maximum file size is 20MB per attachment
6. Images are automatically compressed to save storage space
7. Once uploaded, the file appears in the attachments list with a preview thumbnail
8. Click any attachment to view or download it
9. To remove an attachment, click the delete icon next to the file name
