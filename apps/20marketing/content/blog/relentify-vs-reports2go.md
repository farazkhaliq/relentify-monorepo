Reports2Go has been a familiar name in property inventory software for years, used by independent clerks and letting agents across the UK. It offers a mobile app for creating inventory reports, with templates, photo capture, and PDF generation.

Relentify Inventories is a newer entrant that takes a fresh approach to the same problem — building inventory reports quickly, professionally, and with enough detail to hold up in deposit disputes.

If you are an inventory clerk choosing between the two, or a letting agent evaluating your options, here is a detailed comparison of what each platform offers.

## Overview

Both platforms solve the same core problem: creating structured, photo-rich property inventory reports using a mobile device. But they differ in their approach to pricing, features, and the overall experience.

| Report types | Check-in, check-out, mid-tenancy, interim | Check-in, check-out, mid-tenancy |

| Offline mode | Yes | Yes |

## Report creation

The core workflow is similar in both tools: you move through a property room by room, photograph items and areas, note their condition, and the software compiles everything into a structured PDF.

**Reports2Go** uses a template-based system where you select a property type (flat, house, HMO) and the app pre-populates rooms and items. You can customise these templates, but the initial setup can feel rigid if your properties do not fit the standard categories.

**Relentify** also uses templates but takes a more flexible approach. You can start from a blank canvas or a template, add rooms and items freely, and reorder sections as you go. The interface is designed around speed — fewer taps to capture a photo, add a note, and move on.

For clerks conducting multiple inspections per day, this difference in workflow efficiency adds up. Saving even two or three minutes per room across a five-room property means finishing 10-15 minutes earlier — and across a full day of inspections, that is an extra property you can fit in.

## Condition ratings

Consistent condition ratings make reports more defensible in deposit disputes. Both platforms support condition scales, but the implementation differs.

**Reports2Go** uses a text-based condition description (e.g., "Good", "Fair", "Poor") that clerks select from a dropdown. This is simple and familiar.

**Relentify** uses a standardised numerical scale alongside descriptive terms, making it easier to compare conditions between check-in and check-out objectively. The scale is consistent across all reports, reducing the risk of subjective language causing confusion in adjudication.

## Comparison reports (check-in vs check-out)

When a tenancy ends, the ability to compare check-in and check-out reports side by side is crucial for identifying changes and supporting deposit claims.

**Relentify** builds this comparison into the platform. When you create a check-out report, the check-in data is automatically loaded alongside it. You can compare room by room, item by item, with photos from both inspections displayed together. The comparison report is generated automatically, highlighting differences and changes in condition.

**Reports2Go** supports check-out reports that reference the check-in, but the comparison is less automated. Clerks typically need to manually cross-reference the two reports, which takes more time and increases the risk of missing a detail.

## Offline functionality

Both platforms support offline use, which is essential for inventory clerks working in properties with no Wi-Fi or poor mobile signal.

**Relentify** stores the full report locally on the device and syncs when connectivity is restored. Photos, notes, and condition ratings are all saved locally — you can complete an entire inspection without any connection.

**Reports2Go** also supports offline work, though some users report that certain features (like template downloads) require an initial connection before the inspection begins.

## Support and onboarding

**Reports2Go** has been in the market longer and has an established support system with documentation, email support, and training materials.

**Relentify** offers live chat support, a comprehensive help centre, and onboarding assistance for new users. The interface is designed to be intuitive enough that most clerks can start producing reports within their first session — no formal training required.
